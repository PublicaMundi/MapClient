Data API Documentation

Data API allows users to query any resource that has been added to the catalog using the PublicaMundi Vector Storer extension.

Data can be queried using an instance of Query class.

Available resources can be retrieved using the getResources function.

Details about a resource, its fields and geometry CRS can be obtained by calling describeResource function.

var query = new PublicaMundi.Data.Query('path/to/service/endpoint/');

query.resource('cities', 'table1').
    greater({name : 'h'}, 100).
    orderBy('h', true).
    skip(20).
    take(50);

query.execute({
    success: onSuccess,
    failure: onFailure,
    complete: onComplete
});
