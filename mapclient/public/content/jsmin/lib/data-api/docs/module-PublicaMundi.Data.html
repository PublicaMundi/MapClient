(static) configure(options)

Sets configuration options for all Query instances.
Parameters:
Name Type Description
options object Configuration options.
Properties
Name Type Attributes Default Description
debug boolean <optional>
 false Enables debug mode.
endpoint string <optional>
 Data API service endpoint. Endpoint value can be overriden when creating a new Query instance.
alias object Declares alias names for resources.
Example
PublicaMundi.Data.configure({
     debug: true,
     alias: {
         'roads': 'f62e6867-f74c-4342-9755-976b574ac7f7',
         'cities': 'b6fc004b-07bd-4945-bf34-8ad4a8781c91'
     }
 });
