Class: Query

PublicaMundi.Data.Query

This class provides the core functionality of the Data API. It offers the methods for creating queries and submiting requests. Internally the query is stored as an object. Users can format queries manually and submit JSON messages to the API endpoint*.

*It is recommended that the fluent API is used instead of creating custom queries.

Constructor

new Query(endpoint)

Represents a query.
Parameters:
Name Type Description
endpoint string Data API endpoint. If no endpoint is defined, the default configuration property is used. If API has not been configured current is used.
